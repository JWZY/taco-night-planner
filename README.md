# Taco Night Planner

A fun, interactive app for planning your taco night and assigning ingredients to different people.

## Features

- Add and manage taco ingredients
- Assign ingredients to different players
- Customize with emoji icons
- Responsive design for all devices

## Technologies Used

- Next.js 15
- React 19
- Tailwind CSS
- Radix UI Components

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/JWZY/taco-night-planner.git
cd taco-night-planner
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
